# Sonic Robo Blast 2 Setup Guide

## Required Software

- [Sonic Robo Blast 2](https://www.srb2.org/download/)
- [Latest release of MultiworldGG](https://github.com/MultiworldGG/MultiworldGG/releases). 
- [SL_ArchipelagoSRB2_Vxxx.pk3 from the latest release of the SRB2 mod](https://github.com/GraymonDgt/SRB2Archipelago/releases)

## Setup Guide

- Download SL_ArchipelagoSRB2_Vxxx.pk3 from the Github repository. 

- If you are not using MultiworldGG, also download the apworld file and install it by opening it or moving it to C:\ProgramData\Archipelago\custom_worlds\ 

- Move SL_ArchipelagoSRB2_Vxxx.pk3 into the addons folder of your desired SRB2 installation.

## Joining a Multiworld Game

- After opening the Archipelago/MultiworldGG client, there will be an option labeled "Sonic Robo Blast 2 Client".

- When opening that, a dialogue will open asking for your SRB2 directory, select the same folder as srb2win.exe/gamedata.dat.

- After connecting using the server address and port number, SRB2 will automatically open with the AP mod loaded.
